# Diagramas de classe

> Coloque aqui todas as versões do seu diagrama de classes. Cada versão deve começar com um número de dois dígitos, na ordem crescente.

## v01

```mermaid
classDiagram
    class Produto {
      +String descricao
      +float precoDeCusto
      +float precoDeVenda
      +float margemDeLucro
      +float valorDeImpostos
    }
```
